## 🎯 About

**Super Heroes Rework** is an open-source Minecraft mod that brings superhero powers to modern Minecraft versions (1.20.1+). 

Inspired by the legendary [Fisk's Superheroes](https://www.curseforge.com/minecraft/mc-mods/fisks-superheroes) by FiskFille, this is an **independent, original work** built from the ground up.

> ⚠️ **Note**: This is NOT a port or copy of Fisk's Superheroes. All code, animations, models, textures, and assets are original creations.

---

## ✨ Features

### 🎬 Core Systems

- 🦸 **Hero Framework** - Modular system for creating and managing heroes
- ⚡ **Ability System** - Cooldowns, energy costs, and dynamic abilities
- 🎨 **Custom Animation Engine** - `.frb` format (open source spec)
- 🌐 **Network Sync** - Multiplayer-ready architecture
- 🎮 **Modern HUD** - Energy bars, ability indicators, hero status
